## 项目定位

**产品名称：** 私人助理（Daily Awareness App）

**核心定位：** 语音行为记录与温和提醒助手

**目标用户：**
- 需要规律服药的老年人/慢性病患者
- 希望养成健康生活习惯的人群
- 日常事务繁杂、容易遗忘的上班族

**核心价值：**
1. **语音记录，零门槛** - 按住说话即可记录当前行为，无需手动输入
2. **智能匹配，自动归档** - 时间线记录自动匹配计划事程，无需手动确认
3. **温和提醒，不打扰** - 多级提醒强度可配置，适应不同事项重要性
4. **行为洞察，认识自己** - 分析行为规律，发现习惯模式，提供优化建议

## 技术选型总览

### 客户端技术栈

| 技术领域 | 选型方案 | 说明 |
|---------|---------|------|
| 跨平台框架 | **Flutter 3.x** | 一套代码生成iOS/Android应用 |
| 状态管理 | **Riverpod** | 响应式状态管理，代码简洁 |
| 本地存储 | **SQLite + Hive** | SQLite存结构化数据，Hive存配置 |
| 网络请求 | **Dio + Retrofit** | Dio处理请求，Retrofit生成API客户端 |
| 语音录制 | **flutter_sound** | 支持录音、播放、波形显示 |
| 本地通知 | **flutter_local_notifications** | 到点提醒、自定义提醒样式 |
| 后台任务 | **workmanager** | 定时同步、状态监测等后台任务 |

### 服务端技术栈

| 技术领域 | 选型方案 | 说明 |
|---------|---------|------|
| 后端框架 | **Go / Node.js** | 高性能，适合实时处理 |
| 数据库 | **PostgreSQL** | 关系型数据库，支持复杂查询 |
| 缓存 | **Redis** | 会话缓存、热点数据缓存 |
| 消息队列 | **RabbitMQ / Kafka** | 异步任务处理、提醒调度 |
| 对象存储 | **阿里云OSS / MinIO** | 存储语音文件、用户数据备份 |

### AI服务选型

| 服务类型 | 选型方案 | 说明 |
|---------|---------|------|
| 语音识别（ASR） | **讯飞语音 / 阿里云语音** | 国内方言支持好，延迟低 |
| NLP意图识别 | **讯飞NLP / 阿里云NLP** | 提取行为关键词、时间解析 |
| 对话生成（可选） | **DeepSeek-V3 / Qwen** | 复杂语义理解、自然对话确认 |

### 核心数据流

#### 语音记录流程

```
用户按住说话
    │
    ▼
App录音 + 波形动画反馈
    │
    ▼
上传语音文件到OSS
    │
    ▼
调用ASR服务转文字
    │
    ▼
调用NLP服务提取关键词
    │
    ▼
生成时间线记录（本地+云端）
    │
    ▼
匹配引擎扫描待匹配事程
    │
    ├─ 匹配成功 → 标记事程已完成
    │
    └─ 无匹配 → 仅记录时间线
```

#### 到点提醒流程

```
事程到达计划时间
    │
    ▼
提醒服务触发
    │
    ▼
根据提醒规则生成通知
    │
    ▼
推送服务发送到App
    │
    ▼
App接收通知
    │
    ├─ 用户确认完成 → 调用匹配引擎
    │
    ├─ 用户延后 → 设置下次提醒时间
    │
    └─ 用户跳过 → 标记事程已跳过
    │
    ▼
超时未响应 → Escalating策略触发
    │
    ▼
家属通知（如配置）
```

## MVP功能范围

基于原型文档的优先级定义，MVP阶段实现以下核心功能：

### P0 - 必须实现

| 功能 | 说明 |
|------|------|
| 语音输入 | 按住录音、波形反馈、语音转文字 |
| NLP解析 | 提取行为关键词、时间表达式 |
| 时间线记录 | 自动时间戳、时间线展示、历史检索 |
| 计划事程 | 创建、编辑、查看、状态管理 |

### P1 - 应该实现

| 功能 | 说明 |
|------|------|
| 匹配引擎 | 关键词匹配、时间窗口校验、自动标记 |
| 到点提醒 | 本地通知、语音播报、延后/跳过 |
| 自定义提醒规则 | 按事程配置提醒方式/强度 |
| 无时间事程处理 | 四层策略（推断/浮动/推荐/记录） |

### P2 - 可以延后

| 功能 | 说明 |
|------|------|
| 行为分析汇聚 | 频率分析、异常检测、趋势预测 |
| 常用事程自动提取 | 高频行为识别、自动创建事程 |
| 强制日程 | Escalating强提醒、家属联动 |
