# 第一部分：全局视觉规范

## 1.1 设计原则

本应用面向中老年用户设计，遵循以下核心原则：

| 原则 | 说明 |
|------|------|
| 大字体优先 | 正文 15px 起，关键信息 16-22px，确保 readability |
| 高对比度 | 文字与背景对比度符合 WCAG AA 标准 |
| 触控友好 | 最小点击区域 44x44px，按钮足够大 |
| 单任务聚焦 | 每页只突出一个核心任务，减少认知负担 |
| 即时反馈 | 操作后 200ms 内必须有视觉响应 |
| 防误触设计 | 危险操作需二次确认，重要按钮间距充足 |

# 第二部分：首页

## 2.1 页面概述

首页是应用的核心入口，包含两大视图：
- **时间线视图**：展示用户全天行为记录的时间线列表
- **事程视图**：展示今日待办、已过期、已完成事程的分组列表

用户可通过顶部的 Tab 切换在两个视图间快速跳转。底部提供语音输入和文字输入两种记录方式。
